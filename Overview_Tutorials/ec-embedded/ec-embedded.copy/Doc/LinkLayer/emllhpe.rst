emllHPE is part of |Product| for INtime. emllHPE uses the INtime High-Performance Ethernet (HPE) interface.
The parameters to emllHPE are setup-specific. The function :cpp:func:`CreateLinkParmsFromCmdLineHPE` in :file:`EcSelectLinkLayer.cpp` demonstrates how to initialize the Real-time Ethernet Driver instance.
