Setting up and running EcMasterDemo on SylixOS
**********************************************

Install OS on your hardware with RealEvo-IDE V5.0.4 


How to create the demo application
==================================

#. Start RealEvo-IDE V5.0.4

#. Import EcMasterDemo project

#. Adjust path to Base Project

#. Build

How to run the EcMasterDemo applications
========================================

#. Copy Executable, shared libraries and ENI file to the target

#. Run application 

.. prompt:: bash

    ./EcMasterDemo ...
